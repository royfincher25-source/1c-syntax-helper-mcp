## Qwen Added Memories
- Проект: 1c-syntax-helper-mcp - MCP сервер для парсинга документации 1С. Оптимизирован для больших HBK архивов (до 500MB). Ключевые изменения: 1) Автоматический расчет таймаута извлечения: 600с + 15с/MB сверх 40MB (макс 1 час). 2) LRU кэширование файлов: 256MB память, 10K файлов. 3) Исправлена обработка 7zip exit code 2 (предупреждения вместо ошибок). 4) Используется asyncio.wait_for() для контроля таймаутов. Конфигурация в constants.py: HBK_EXTRACT_TIMEOUT_BASE=600, HBK_EXTRACT_TIMEOUT_PER_MB=15, HBK_EXTRACT_TIMEOUT_MAX=3600, MEMORY_CACHE_LIMIT_MB=256, MEMORY_CACHE_LIMIT_FILES=10000, MAX_FILE_SIZE_MB=500.
