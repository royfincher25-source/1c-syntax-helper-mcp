## 🔍 Технические детали

### Архитектура кэша:

```
InMemoryCache (max_size=1000, default_ttl=300s)
├── OrderedDict для хранения записей
├── asyncio.Lock для потокобезопасности
├── Фоновая задача очистки (interval=60s)
└── CacheStats для метрик
```

### Формат ключей:

```
search:{query}:{limit}           # Поисковые запросы (TTL: 5 мин)
syntax:{object}:{element}:{ex}   # Детальная информация (TTL: 1 час)
```

### LRU Eviction:

```python
while len(cache) >= max_size:
    oldest_key = next(iter(cache))  # Первый ключ - самый старый
    del cache[oldest_key]
    stats.evictions += 1
```
