## 🔍 Технические детали

### Multi-stage структура:

```dockerfile
# Этап 1: Builder
FROM python:3.11-slim as builder
- Установка системных зависимостей (gcc, python3-dev)
- Создание venv
- Установка Python зависимостей

# Этап 2: Production
FROM python:3.11-slim as production
- Только runtime пакеты (curl, p7zip-full, unzip)
- Копирование venv из builder
- Не-root пользователь
- Health check

# Этап 3: Development (опционально)
FROM production as development
- Dev инструменты (git, vim)
- Dev зависимости
- Auto-reload
```

### .dockerignore исключения:
- Git: `.git/`, `.gitignore`
- Python: `__pycache__/`, `*.pyc`, `.pytest_cache/`
- IDE: `.idea/`, `.vscode/`
- Конфиги: `*_config.json`, `*.md`
- Данные: `data/hbk/*.hbk`
