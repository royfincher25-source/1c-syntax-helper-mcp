## 🎯 Как использовать

### В новой сессии:

1. **Открой шпаргалку:**
   ```
   docs/plans/OPTIMIZATION_QUICKSTART.md
   ```

2. **Найди текущую задачу:**
   - Посмотри на статус фаз
   - Выбери следующую задачу из списка

3. **Скопируй чеклист:**
   ```markdown
   **1.1 Docker Multi-stage Build** (2 часа)
   - [ ] Изучить текущий Dockerfile
   - [ ] Создать multi-stage версию
   - [ ] Протестировать сборку
   - [ ] Замерить размер образа
   ```

4. **Начни работу:**
   ```
   Продолжаю оптимизацию по плану docs/plans/2026-03-05-optimization-roadmap.md
   Текущая задача: 1.1 Docker Multi-stage Build
   
   Создаю todo list:
   - [ ] Задача 1
   - [ ] Задача 2
   ```

5. **Задокументируй прогресс:**
   - Отметь выполненные пункты ✅
   - Создай отчёт о сессии (шаблон в шпаргалке)
   - Закоммить изменения
